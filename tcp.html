TCP Socket Programming and Packet Capture

For this project, I built a simple client–server application in Python to explore TCP socket programming and real-time network communication.

The client and server exchange echo messages using Python’s socket library, and I used Wireshark to capture and analyze network packets. This helped visualize how application-layer messages correspond to transport-layer packets.

The project strengthened my understanding of TCP/IP protocols, multi-threaded communication, and network debugging.
