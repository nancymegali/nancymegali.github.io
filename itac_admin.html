ITAC Admin Portal (Full-Stack Web Application)

For this project, I helped build a full-stack admin portal for the Industrial Training and Assessment Centers (ITAC) program at Rutgers University. The portal allows program administrators to manage center data, reports, and research documentation in one place.

I worked on the site’s frontend using React and set up the backend with MySQL for reliable data storage. The app runs in a Dockerized environment to keep development and production consistent, and it integrates with Google Cloud Platform (GCP) to securely store and retrieve PDFs.

I collaborated with a team of developers to design and build dashboard modules such as Student Statistics, Budget Tracking, and Quarterly Reporting. My focus was on creating reusable components, connecting APIs cleanly, and making the interface easy for admins to use.
