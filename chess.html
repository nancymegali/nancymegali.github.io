Chess Project

For this project, I built a fully functional chess game in Java for my Systems Programming course at Rutgers University.

I implemented all the core game logic, including move validation, check and checkmate detection, castling, en passant, and pawn promotion. The game runs in a two-player mode with a text-based interface for gameplay.

This project focused heavily on object-oriented design, using inheritance and data structures to keep the code modular, organized, and easy to maintain. I also spent time testing each rule thoroughly to make sure every move worked as expected.
